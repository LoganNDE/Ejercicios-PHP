# 2️⃣ Intercambiar

Crea una función llamada `intercambia` que reciba dos parámetros numéricos por referencia y los intercambie. Si recibe `$a` y `$b`, tras ejecutar la función `$a` tendrá el valor de `$b` y viceversa.
